## Sommario

Slide introduttive del corso Advanced Computer Programming tenuto dal Prof. Luigi De Simone presso l'Università degli Studi di Napoli Federico II. Il corso copre due macro-aree: Python avanzato e Java. Per Python si affrontano basi, OOP, concorrenza, networking, middleware (RPC, MOM), framework web (Flask) e NoSQL. Per Java si trattano multithreading, networking e middleware. Il corso include anche aspetti di deployment (Docker, Kubernetes). L'esame prevede una prova al computer seguita da un colloquio orale.

## Punti chiave

1. Il corso si chiama **Advanced Computer Programming** (ACP)
2. Docente: **Prof. Luigi De Simone**, UniNA Federico II
3. Due macro-aree: **Python** e **Java**
4. Python copre: basi, OOP, concorrenza (threading, multiprocessing, asyncio), networking, middleware (RPC/gRPC, MOM/STOMP), Flask, NoSQL (MongoDB)
5. Java copre: multithreading, networking, middleware
6. Deploy: Docker, Kubernetes
7. Esame: **prova al computer** + **colloquio orale**
8. Libri: Think Python (Downey), Gaddis, Deitel, Thinking in Java (Eckel)
